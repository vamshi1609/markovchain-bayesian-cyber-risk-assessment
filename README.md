# 🔐 Cyber Risk Assessment System using Markov Chains & Bayesian Networks

---

## 📌 Overview
This project is a machine learning-based cybersecurity risk analysis system that combines **Markov Chains** and **Bayesian Networks** to model system behavior and predict risk levels dynamically.

---

## ⚙️ How It Works

1. System states are defined (Safe, Warning, Critical)
2. Markov Chain models transitions between states
3. Bayesian Network calculates probability of risk conditions
4. Final output generates a cyber risk score

---

---

## 🚀 Features

- Cyber risk prediction model  
- Probabilistic decision-making system  
- State transition modeling using Markov Chains  
- Risk scoring using Bayesian inference
